# New Screens in v1.1

Version 1.1 introduces several new screens focused on social features and enhanced user experience:

  1. Onboarding View: A streamlined introduction for new users with options for guest mode or account creation.

  2. Bestie Matching View: A dedicated screen for finding and connecting with users who have similar music taste.

  3. History View: A timeline view showing how rankings have changed over time.

  4. Enhanced Sidebar Panel: The sidebar now includes three main tabs:

    • Your Journal (personal rankings and reviews)
    • Community (reviews and comments from other users)
    • Besties (connected users with similar taste)

  5. Reddit Verification Flow: A process for verifying user identity through Reddit to build trust in the community.

Each of these screens is detailed in the specific screen documentation sections.
