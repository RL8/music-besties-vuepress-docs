# Components Overview

This section provides a big picture overview of all the components used in the app for version 1.1.

# Component Hierarchy

The application is structured around these key components, which are implemented in Vue.js:

Figure: Component Hierarchy

Tree diagram showing the Vue.js component hierarchy from App.vue down to individual UI elements. This diagram reflects the complete application structure for version 1.1 including new social features.

# Primary Components

  • App.vue: The root component that manages routing and global state, including theme initialization
  • DashboardView.vue: Main entry point displaying the Era Cards grid with era-specific styling
  • EditRankingsView.vue: Interface for selecting and ranking items with themed elements
  • SidebarPanel.vue: Context-sensitive panel with three main tabs: Your Journal, Community, and Besties
  • BestieMatchingView.vue: New in v1.1 - Interface for finding and connecting with similar users
  • OnboardingView.vue: New in v1.1 - First-time user experience with introduction and setup
  • NavigationSystem.vue: Global navigation system accessible from all views

# Supporting Components

  • EraCard.vue: Reusable component for era display on the dashboard, styled with era-specific colors and fonts
  • HistoryIndicator.vue: New in v1.1 - Visual indicator showing ranking history is available
  • SelectionTabs.vue: Tab navigation for different ranking categories, with active tabs styled based on era theme
  • ItemChip.vue: Interactive component for selection functionality, using era-specific styling for selected state
  • CooldownTimer.vue: New in v1.1 - Timer displaying the cooldown period until next ranking save
  • RankingsList.vue: Component for displaying ordered rankings with era-themed indicators
  • SongCard.vue: New in v1.1 - Compact display of songs with rank, rating, and comment count
  • ReviewDisplay.vue: Form component for ratings and comments, styled according to the current era theme
  • CommentSection.vue: New in v1.1 - Sequential blog-like commenting system for songs
  • CommunityReviewsList.vue: New in v1.1 - List of reviews from other users
  • BestieMatchList.vue: New in v1.1 - Display of connected and discoverable users with similar tastes
  • ShareOptions.vue: New in v1.1 - Interface for sharing rankings and reviews on social media

# Navigation Components

  • NavMenu.vue: Menu component with navigation links
  • AboutView.vue: View component for application information
  • ProfileView.vue: View component for user profile, enhanced in v1.1 with social features
  • SettingsView.vue: View component for user settings, including theme selection options
  • TermsView.vue: View component for legal information

# Theme Integration

All components are designed to work with the era-specific theming system documented in the Theme System section. Components use CSS variables (--era-primary, --era-secondary, --era-font) to apply consistent styling based on the selected era.

Key themed elements include:

  • Era-specific card styling with themed borders and backgrounds
  • Themed active tab indicators
  • Era-specific selection indicators
  • Themed buttons and interactive elements
  • Era-specific fonts for themed content

# New Components in v1.1

Version 1.1 introduces several new components focused on social features and user experience:

  1. Social Components:

    • BestieMatchingView.vue - Find users with similar music taste
    • BestieMatchList.vue - Display potential and connected "besties"
    • RedditVerification.vue - Verify user identity through Reddit
    • ShareOptions.vue - Share rankings and reviews on social media

  2. Enhanced User Experience:

    • OnboardingView.vue - Streamlined first-time user experience
    • WelcomeCarousel.vue - Introduction slides for new users
    • GuestModeButton.vue - Quick start option without account creation
    • ComparisonFlow.vue - Entry point for shared comparison links

  3. History and Tracking:

    • HistoryIndicator.vue - Visual indicator for available ranking history
    • CooldownTimer.vue - Timer displaying cooldown period for ranking saves
