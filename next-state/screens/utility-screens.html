# Settings and Utility Screens

This section covers the utility screens in the Music Besties application v1.1, including Settings, Profile, About, and Terms/Privacy views. These screens have been enhanced with privacy controls and Reddit account verification features.

# Settings View

The Settings View allows users to configure application preferences, privacy options, and account verification.

# Screen Overview

Purpose: Provide user control over application behavior, appearance, data management, privacy settings, and account verification

Primary Components:

  • SettingsHeader.vue (title and back button)
  • SettingsGroup.vue (category grouping)
  • SettingsToggle.vue (boolean settings)
  • SettingsRadio.vue (option selection)
  • SettingsButton.vue (action buttons)
  • PrivacyControls.vue (visibility settings for public profile)
  • RedditVerification.vue (Reddit account connection and verification)

# New Privacy Features

Version 1.1 introduces enhanced privacy controls that allow users to:

  • Toggle public profile visibility
  • Control which aspects of their profile are visible to others:
    • Rankings visibility
    • Reviews visibility
    • Besties connections visibility
  • Manage data sharing preferences
  • Verify their identity through Reddit integration

# Reddit Verification

The Reddit verification feature allows users to:

  1. Connect their Reddit account to Music Besties
  2. Verify their identity through a secure OAuth flow
  3. Display a verification badge on their profile
  4. Build trust within the community
  5. Participate in verified-only discussions

# Profile View

The Profile View displays user information, statistics, and sharing options.

# Screen Overview

Purpose: Provide a comprehensive view of the user's activity, preferences, and social connections

Primary Components:

  • ProfileHeader.vue (title, back button, and profile image)
  • PublicLinkDisplay.vue (shareable profile link)
  • UserStatistics.vue (activity metrics)
  • ActivityHistory.vue (recent actions)
  • BestiePreview.vue (top matches)
  • PreferenceDisplay.vue (user preferences)

# Social Features

Version 1.1 enhances the Profile View with social features:

  1. Public Profile Link: Shareable URL for the user's profile
  2. Bestie Matches: Display of users with similar music preferences
  3. Activity Timeline: Record of recent ranking and review activities
  4. Verification Badge: Indicator of Reddit account verification status

# New in v1.1

The utility screens have been enhanced in v1.1 with the following features:

  1. Enhanced Privacy Controls: Granular control over what information is shared
  2. Reddit Verification: Account verification through Reddit integration
  3. Social Sharing Options: Easy sharing of profiles and rankings
  4. Bestie Matching Integration: View and manage bestie connections
  5. Activity Timeline: Track recent actions and changes
  6. Theme Integration: Components now use the era-specific theming system
  7. Expanded Data Management: Options to export and reset different types of data
