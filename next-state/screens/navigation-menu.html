# Navigation Menu

The Navigation Menu provides global navigation to all major sections of the Music Besties application v1.1. It slides in from the left when the user taps the hamburger menu icon. In v1.1, it has been enhanced with account options for guest users and a new "Your Besties" section for the social matching feature.

# Screen Overview

Purpose: Provide consistent access to all main sections of the application from any screen, with appropriate options based on authentication state

Primary Components:

  • NavMenu.vue (container and animation)
  • NavMenuItem.vue (individual menu items)
  • NavHeader.vue (app branding and close button)
  • AccountOptions.vue (login/register buttons for guest users)

# Component Details

# NavMenu.vue

The container component that handles the slide-in animation and overall menu structure.

Properties:

  • isOpen (Boolean): Whether the menu is currently visible
  • currentRoute (String): The current application route
  • isAuthenticated (Boolean): Whether the user is currently logged in
  • isGuest (Boolean): Whether the user is in guest mode

Events:

  • @close: Emitted when the close button is clicked
  • @navigate: Emitted when a menu item is clicked, with the route name

Animation:

  • Smooth slide-in from left with overlay backdrop
  • Transition duration: 300ms
  • Easing: ease-in-out

# NavMenuItem.vue

Individual menu item component with icon and text.

Properties:

  • icon (String): Icon name or emoji
  • text (String): Menu item text
  • route (String): Route name for navigation
  • isActive (Boolean): Whether this item represents the current route
  • disabled (Boolean): Whether the item is clickable

Events:

  • @click: Emitted when the item is clicked, with the route name

Styling:

  • Active item has highlight background and left border
  • Hover state with subtle background change
  • Disabled items have reduced opacity
  • Icon and text are aligned with consistent spacing

# NavHeader.vue

The header component with app branding and close button.

Properties:

  • appName (String): The application name
  • logoUrl (String): URL to the app logo image

Events:

  • @close: Emitted when the close button is clicked

# AccountOptions.vue

The AccountOptions component provides login and registration options for guest users.

Properties:

  • isAuthenticated (Boolean): Whether the user is currently logged in
  • showButtons (Boolean): Whether to display the login/register buttons
  • isGuest (Boolean): Whether the user is in guest mode

Events:

  • @login: Emitted when the login button is clicked
  • @register: Emitted when the create account button is clicked
  • @logout: Emitted when the logout button is clicked

# Navigation Structure

The Navigation Menu provides access to the following sections:

  1. Dashboard: Main entry point with era cards grid
  2. Profile: User profile with statistics and preferences
  3. Your Besties: New in v1.1 - Social matching and connections
  4. Settings: Application preferences and account settings
  5. About: Application information and version details
  6. Terms & Privacy: Legal information and privacy policy

# User Interactions

Interaction Result Implementation
Tap hamburger icon Opens navigation menu Emits @open event
Tap close button Closes navigation menu Emits @close event
Tap menu item Navigates to section Emits @navigate event with route
Tap Login Opens login form Emits @login event
Tap Create Account Opens registration form Emits @register event
Tap outside menu Closes navigation menu Emits @close event

# Authentication States

The Navigation Menu adapts to different authentication states:

  1. Guest Mode:

    • Shows Login and Create Account buttons
    • All sections are accessible
    • Profile shows limited information
    • Your Besties shows limited functionality

  2. Authenticated User:

    • Hides Login and Create Account buttons
    • Shows username or profile picture
    • All sections are fully accessible
    • Your Besties shows full functionality

# Accessibility Features

  • All interactive elements have appropriate ARIA labels
  • Color contrast meets WCAG 2.1 AA standards
  • Keyboard navigation is fully supported
  • Focus is trapped within the menu when open
  • Escape key closes the menu

# New in v1.1

The Navigation Menu has been enhanced in v1.1 with the following features:

  1. Your Besties Section: New navigation item for the social matching feature
  2. Guest Mode Support: Adapted UI for users in guest mode
  3. Authentication Options: Login and registration buttons for guest users
  4. Improved Animation: Smoother transitions with backdrop overlay
  5. Theme Integration: Components now use the application theming system
