# Edit Rankings View

The Edit Rankings View in Music Besties v1.1 provides an interface for users to select and rank items within a specific music era, with new options for quick ranking or comprehensive organization.

# Screen Overview

Purpose: Allow users to create and modify their rankings of music eras and songs with flexible options for different user needs

Key Enhancements in v1.1:

  • New Quick Rank feature for ranking just top 3 eras and top 3 songs per era
  • Selection modal when clicking "Edit Rankings" to choose ranking experience
  • Enhanced sorting options that affect all tabs and save user preferences
  • History tracking to view how rankings have changed over time
  • Improved drag-and-drop interface with clearer visual feedback

# Quick Rank Feature

# Access Method

  • User clicks "Edit Rankings" on dashboard or in sidebar panel
  • Popup appears with 2 options:
    • "Quick rank": Limited functionality with clear messaging
    • "Gotta rank 'em all!": Full ranking experience

# Quick Rank Limitations

  • Only allows ranking of top 3 eras
  • Only allows ranking of top 3 songs per era
  • Clear messaging about limitations
  • Option to upgrade to full ranking at any time

# Enhanced Sorting Options

# Sorting Capabilities

  • Sort by alphabetical order (A-Z)
  • Sort by release date (oldest to newest)
  • Sort by user ranking (if available)
  • Sort by popularity (based on community rankings)

# Persistence

  • User's preferred sorting method is saved
  • Sorting preference applies across all tabs
  • Clear visual indication of current sort method

# Component Details

# RankingSelectionModal.vue

New in v1.1, this component displays the options for quick rank or full ranking.

Properties:

  • visible (Boolean): Whether the modal is visible
  • options (Array): Array of ranking options with descriptions

Events:

  • @select: Emitted when an option is selected
  • @close: Emitted when the modal is closed

# RankingList.vue

Component for displaying and reordering ranked items.

Properties:

  • items (Array): List of items to rank
  • rankedItems (Array): Currently ranked items
  • maxItems (Number): Maximum number of items that can be ranked (for Quick Rank)
  • dragEnabled (Boolean): Whether drag-and-drop is enabled

Events:

  • @reorder: Emitted when items are reordered
  • @add: Emitted when an item is added to the ranked list
  • @remove: Emitted when an item is removed from the ranked list

# SortingControls.vue

Component for selecting and applying sorting options.

Properties:

  • options (Array): Available sorting options
  • selectedOption (String): Currently selected sorting option
  • persistent (Boolean): Whether to save the selection for future visits

Events:

  • @change: Emitted when the sorting option changes

# HistoryView.vue

New in v1.1, this component displays the history of ranking changes.

Properties:

  • history (Array): Array of historical ranking snapshots
  • currentRanking (Array): Current ranking for comparison

Events:

  • @restore: Emitted when a historical ranking is selected for restoration

# User Interactions

Interaction Result Implementation
Tap "Edit Rankings" Opens ranking selection modal Displays modal with quick rank and full ranking options
Select "Quick rank" Opens ranking interface with limitations Initializes RankingList with maxItems=3
Select "Gotta rank 'em all!" Opens full ranking interface Initializes RankingList with no item limit
Drag item Reorders the item in the list Uses HTML5 drag-and-drop API with visual feedback
Tap sort option Changes the sort order of items Emits @change event with new sort option
Tap "Save Rankings" Saves current rankings Sends ranking data to API and updates history
Tap "Restore This Ranking" Reverts to historical ranking Emits @restore event with selected ranking data

# Accessibility Considerations

  • Drag-and-drop functionality has keyboard alternatives
  • All interactive elements have appropriate ARIA labels
  • Color contrast meets WCAG 2.1 AA standards
  • Focus management for modal dialogs
  • Screen reader announcements for ranking changes
