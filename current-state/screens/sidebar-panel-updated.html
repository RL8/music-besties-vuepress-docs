# Sidebar Panel

The Sidebar Panel is a context-sensitive panel that slides in from the right side of the screen, providing detailed information and interaction options for a specific music era. The panel implements era-specific styling as documented in the Design Foundations and Theme System sections.

# Component Details

# SidebarHeader.vue

The header component with era name, emoji, and close button.

Properties:

  • era (Object): Era information including name and ID
  • showBackButton (Boolean): Whether to show the back button

Events:

  • @close: Emitted when the close button is clicked
  • @back: Emitted when the back button is clicked

# TabNavigation.vue

The tab navigation component for switching between Rankings and Review tabs.

Properties:

  • tabs (Array): List of tab objects with id, title, and active properties
  • activeTabId (String): ID of the currently active tab

Events:

  • @tabChange: Emitted when a tab is clicked, with the new tab ID

# RankingsList.vue

Component for displaying the ordered list of ranked items.

Properties:

  • items (Array): List of ranked items with id, title, and rank properties
  • totalCount (Number): Total number of available items
  • editable (Boolean): Whether the list can be reordered

Events:

  • @reorder: Emitted when items are reordered, with the new order
  • @remove: Emitted when an item is removed from the list

# ReviewDisplay.vue

Component for displaying and editing reviews.

Properties:

  • review (Object): Review data including rating and text
  • editable (Boolean): Whether the review can be edited

Events:

  • @edit: Emitted when the edit button is clicked
  • @save: Emitted when a review is saved

# NotesList.vue

Component for displaying and managing notes.

Properties:

  • notes (Array): List of note objects with id, text, and timestamp properties
  • editable (Boolean): Whether notes can be added or edited

Events:

  • @add: Emitted when the add note button is clicked
  • @edit: Emitted when a note is edited
  • @delete: Emitted when a note is deleted

# User Interactions

Interaction Result Implementation
Tap close button Closes sidebar panel Emits @close event
Tap back button Returns to previous view Emits @back event
Tap tab Switches between tabs Emits @tabChange event with tab ID
Tap Edit Review Opens review edit form Emits @edit event
Tap Add Note Opens note input form Emits @add event
Drag ranked item Reorders the item in the list Emits @reorder event with new order
Tap remove icon Removes item from rankings Emits @remove event with item ID

# Theme Integration

The Sidebar Panel components use era-specific theming to maintain visual consistency:

  • Header background uses a light version of the era's primary color
  • Era name uses the era's font family
  • Tab indicators use the era's primary color
  • Rating stars use the era's secondary color

# Accessibility Considerations

  • All interactive elements have appropriate ARIA labels
  • Color contrast meets WCAG 2.1 AA standards
  • Keyboard navigation is fully supported
  • Screen reader announcements for tab changes
