# Design Foundations

The foundational elements of the Music Besties design system define the basic visual language of the application. These elements include colors, typography, spacing, effects, and era-specific design specifications.

# Era-Specific Design System

Music Besties features a comprehensive era-based design system that allows for consistent theming across the application. Each era has its own color palette, typography, and associated emoji.

# Iconography

The application primarily uses emoji as icons, with each music era having its own associated emoji. This approach provides consistent visual identifiers without requiring additional image assets.

# Theme Variables

The application uses CSS variables to apply era-specific styling throughout the app:

:root {
    --era-primary: #008080;    /* Default primary color */
    --era-secondary: #B2DFDB;  /* Default secondary color */
    --era-font: 'Outfit', sans-serif;  /* Default font */
}

These variables are updated dynamically based on the selected era using the theme utility system.
